# 🌸 Habitus — Minimal Habit Tracker

A clean, minimalistic habit tracking application designed to help users build consistency through a simple **21-day habit challenge system**.

---

## ✨ Features

* 📝 Add and manage daily habits
* ✅ Mark habits as completed
* 📅 21-day habit tracking system
* 📊 Progress tracking with streak visualization
* 💾 Data persistence using localStorage
* 🎨 Minimal, aesthetic UI
* 📱 Fully responsive (mobile + desktop)

---

##  Tech Stack

**Frontend:**

* HTML
* CSS
* JavaScript

**Backend (separately implemented):**

* Python
* MySQL

---

##  How to Run

```bash
git clone https://github.com/Gunika-Kharbanda/Habitus-.git
```

* Open the folder
* Run `index.html` in browser
  **OR** use Live Server

---

##  Project Concept

Habitus is based on the principle that **consistent small actions over 21 days can help build long-term habits**.
The application provides a distraction-free and calming environment to track daily progress.
